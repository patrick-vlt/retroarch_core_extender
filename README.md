# Usage
- Create venv
```bash
sudo apt-get install -f -y python3-venv
python3 -m venv venv
```
- Activate the venv
```bash
# Unix / MacOS
source venv/bin/activate
# Windows
myenv\Scripts\activate
```
- Setup your playlists.json file
- Run the scraper
```python
python scrape_games_from_igdb.py.py
```
